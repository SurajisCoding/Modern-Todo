Install in 3 steps

Get it on your phone.

1

Download the APK

Tap the "Download for Android (.APK)" button above. The file will download directly to your phone's storage.

2

Allow "Install Unknown Apps"

Android 8+: go to Settings → Apps → Special App Access → Install Unknown Apps, find your browser (e.g. Chrome), and enable it.

Android 7 and below: go to Settings → Security → Unknown Sources and toggle it on. You can switch it off again after installing.

3

Open the file and install

Open your file manager or the notification, tap the downloaded .apk, and hit Install. Modern Todo will appear in your app drawer instantly.

No. Modern Todo stores everything in device-local AsyncStorage and makes zero network requests. It doesn't need camera, contacts, location, notifications, or any sensitive permission. Your tasks never leave your phone.

Data is stored only in AsyncStorage on your device using the key @modern_todo_tasks. There is no cloud sync, no backup service, and no external server. If you uninstall the app, your tasks will be erased. A local export/import feature may be added in a future version.
